About Me

Hi, I'm Natanel Roizenman! Welcome to my simplified portfolio site.

I'm a computer engineering student at the University of Waterloo with a passion for embedded systems, low‑level programming and anything tech. In my spare time I love tinkering with microcontrollers, building small operating systems and exploring the intersection of hardware and software.

Projects

Projects

Here are some of my public projects. These span scripting, tooling, game development and embedded firmware, and all of them are available on my GitHub (nroize).

  • NatanelOS: this, an interactive portfolio modeled to look like a browser‑based operating system. It features a command‑line terminal, windowed UI and even a virtual file system to explore my work.
  • Mosylite: a lightweight Python wrapper for the Mosyle Manager API that helps integrate Apple device management into automation scripts.
  • AimTrainer: a simple aim training game written in Java to improve mouse accuracy and reaction time.
  • PyEdsby: a Python library for building integrations with the Edsby Student Information System.
  • Automated birdfeeder: embedded firmware and supporting code for an automated bird feeder project built using microcontrollers and sensors.
  • CFX‑800: a replacement PCB for the Casio DBC‑611 watch that adds CFX‑400‑like capabilities (e.g. scientific calculations, logarithms, etc.).

All of these projects and more can be found on my GitHub profile.
